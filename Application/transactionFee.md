# ChamaPay Fee Structure 

## 1. Transfer Fees (USDC internal transfers)

Transfers are between ChamaPay users. Fee is charged in USDC.

| Amount (USDC) | Fee (USDC) |
|---------------|-------------|
| 0.01 - 1.00   | 0.01        |
| 1.01 - 10.00  | 0.05        |
| 10.01 - 50.00 | 0.10        |
| 50.01 - 100.00| 0.25        |
| 100.01 - 500.00 | 0.50      |
| 500.01 - 1000.00 | 0.75     |
| > 1000.00     | 1.00 (max)  |

**Rules:**
- Minimum transfer amount: 0.01 USDC
- No maximum transfer amount
- Maximum transfer fee cap: 1 USDC

## 2. Withdrawal to M-PESA Fees (Kshs)

Withdrawal from ChamaPay balance to M-PESA.

| Amount (Kshs) | Fee (Kshs) |
|---------------|-------------|
| 100 - 500     | 5           |
| 501 - 1,000   | 10          |
| 1,001 - 2,500 | 15          |
| 2,501 - 5,000 | 25          |
| 5,001 - 10,000| 40          |
| 10,001 - 20,000 | 60        |
| 20,001 - 35,000 | 100       |
| 35,001 - 50,000 | 150       |
| 50,001 - 70,000 | 220       |
| 70,001 - 100,000 | 300 (max) |

**Rules:**
- Minimum withdrawal: 100 Kshs
- Maximum per transaction: 100,000 Kshs
- Maximum withdrawal fee cap: 300 Kshs
